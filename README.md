# Simulador PIL Andina - Método Símplex

Aplicación web interactiva para la resolución de problemas de optimización lineal utilizando el **Método Símplex (Gran M)**.

## Características

1. **Caso de Estudio (PIL Andina)**: Análisis situacional y resolución paso a paso del modelo de producción de lácteos bajo diferentes escenarios de crisis (Mayo 2026).
2. **Interactividad Dinámica**: Gráficos interactivos, tablas paso a paso, análisis de sensibilidad en tiempo real y selectores de escenario "What-If".
3. **Calculadora Libre**: Una herramienta independiente (`calculadora.html`) que permite definir matrices Símplex (Max/Min, ≤, ≥, =) de cualquier tamaño y generar soluciones iterativas al instante.

## Uso

1. Abre `index.html` en tu navegador para ver el estudio de caso interactivo y el selector de escenarios.
2. Desde la barra de navegación, accede a la **Calculadora Libre** para ingresar tus propios problemas de programación lineal.
3. No requiere instalación ni dependencias externas (usa Chart.js desde CDN).
